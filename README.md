# AI驱动的开发实践：终端数字猜谜游戏

## 项目简介
本项目是基于 C++17 与 FTXUI 终端 UI 库开发的控制台数字猜谜小游戏，属于 C++ 课程实践作业，核心代码由 AI 辅助生成与逻辑设计。
程序在纯终端内实现带色彩排版的可视化界面，完整实现猜数字游戏逻辑：随机生成 1~100 目标数字、输入合法性校验、大小提示、猜测次数统计、游戏结算界面，支持非数字输入容错、区间越界提示。
## 项目结构
ftxui-game/
├── CMakeLists.txt   # CMake构建配置文件，管理C++标准、FTXUI库链接、可执行文件生成
├── main.cpp         # 程序主逻辑，包含随机数生成、FTXUI界面渲染、游戏循环、输入处理
└── README.md        # 项目说明文档




运行程序
编译完成后，在build/Debug或build/Release目录下执行 ai-ftxui-game.exe 启动游戏：
程序自动生成 1~100 随机数字，终端弹出可视化游戏界面；
在输入框输入数字回车提交，程序会提示「太大 / 太小 / 输入非法 / 超出区间」；
猜中目标数字后，展示结算界面，统计总猜测次数；
按下回车键退出程序。
功能特性
界面美化：文字居中、分栏分割线、区分提示文字色彩（蓝色标题、黄色提示、绿色结算）；
输入容错：捕获非数字输入异常，提示用户重新输入；
数值校验：限制输入范围 1~100，越界给出对应提示；
游戏状态留存：实时展示当前累计猜测次数；
结算页面：猜对后单独渲染结束界面，展示完整通关信息。
